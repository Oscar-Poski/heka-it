# ¿Qué es la administración de sistemas?

La **administración de sistemas** es la disciplina encargada de **instalar, configurar, mantener y supervisar sistemas informáticos**.

Un administrador de sistemas se encarga de garantizar que los sistemas:

- funcionen correctamente
- sean seguros
- estén disponibles cuando se necesitan
- tengan buen rendimiento

En entornos Linux, estas tareas suelen realizarse principalmente desde la **terminal**.

---

# Qué hace un administrador de sistemas

Las responsabilidades de un administrador de sistemas pueden incluir:

- instalar sistemas operativos
- configurar servicios del sistema
- administrar usuarios y permisos
- monitorear el uso de recursos
- actualizar software
- solucionar problemas del sistema

En servidores Linux, muchas de estas tareas se realizan **de forma remota usando SSH**.

---

# Administración en entornos reales

En empresas y organizaciones, los sistemas Linux se utilizan para ejecutar:

- servidores web
- bases de datos
- aplicaciones empresariales
- infraestructura cloud
- servicios de red

Los administradores se encargan de mantener estos sistemas **estables y seguros**.

---

# Automatización y herramientas

La administración moderna de sistemas también incluye el uso de herramientas para automatizar tareas.

Por ejemplo:

- scripts de Bash
- herramientas de monitoreo
- gestores de configuración
- plataformas de infraestructura cloud

Esto permite administrar **muchos servidores de forma eficiente**.

---

# Administración preventiva

Un buen administrador de sistemas no solo reacciona ante problemas.

También realiza tareas preventivas como:

- monitorear recursos del sistema
- revisar logs
- aplicar actualizaciones de seguridad
- realizar copias de seguridad

Estas prácticas ayudan a evitar fallos en el sistema.

---

# Linux en administración de sistemas

Linux es uno de los sistemas más utilizados en administración de sistemas porque:

- es estable
- es flexible
- permite automatización
- funciona bien en servidores

Por esta razón, muchas infraestructuras modernas dependen de sistemas Linux.

---

# Idea clave de esta lección

La administración de sistemas consiste en mantener sistemas informáticos funcionando de forma segura, eficiente y confiable.

---

# Repaso

- La administración de sistemas mantiene sistemas funcionando correctamente.
- Incluye instalación, configuración y monitoreo.
- Los administradores gestionan usuarios, servicios y recursos.
- Linux es ampliamente utilizado en servidores.
- Muchas tareas se realizan mediante terminal y automatización.
