# Linux en servidores

Linux es uno de los sistemas operativos más utilizados en **servidores de todo el mundo**.

Gran parte de la infraestructura de internet funciona sobre servidores Linux.

Esto incluye:

- sitios web
- aplicaciones en línea
- plataformas cloud
- bases de datos
- servicios de red

Comprender cómo se utiliza Linux en servidores es fundamental para entender su importancia en la industria tecnológica.

---

# Qué es un servidor

Un **servidor** es una computadora que proporciona servicios a otras computadoras en una red.

Estas computadoras que solicitan servicios se llaman **clientes**.

Por ejemplo:

- un navegador web solicita páginas a un servidor web
- una aplicación solicita datos a un servidor de base de datos
- un usuario se conecta remotamente a un servidor mediante SSH

Los servidores suelen estar diseñados para funcionar **de forma continua y confiable**.

---

# Por qué Linux es tan utilizado en servidores

Linux domina el mundo de los servidores por varias razones.

Entre ellas:

**Estabilidad**

Los sistemas Linux pueden funcionar durante largos periodos sin reiniciarse.

---

**Seguridad**

El modelo de permisos y la arquitectura del sistema ayudan a mantener sistemas protegidos.

---

**Flexibilidad**

Linux puede adaptarse a diferentes tipos de servidores y cargas de trabajo.

---

**Automatización**

Linux permite automatizar tareas fácilmente mediante scripts y herramientas administrativas.

---

**Software libre**

Muchas distribuciones Linux son de código abierto, lo que facilita su adopción en empresas y organizaciones.

---

# Tipos de servidores que usan Linux

Linux se utiliza en muchos tipos de servidores.

Por ejemplo:

**Servidores web**

Entregan páginas y aplicaciones web.

Ejemplos de software:

- Apache
- Nginx

---

**Servidores de bases de datos**

Almacenan y gestionan información.

Ejemplos:

- MySQL
- PostgreSQL

---

**Servidores de aplicaciones**

Ejecutan aplicaciones empresariales o servicios web.

---

**Servidores de archivos**

Permiten compartir archivos dentro de una red.

---

# Linux en internet

Una gran parte de los sitios web del mundo funcionan sobre servidores Linux.

Esto incluye:

- plataformas de comercio electrónico
- redes sociales
- servicios de streaming
- aplicaciones empresariales

Linux es una base importante de la infraestructura de internet.

---

# Administración de servidores Linux

Administrar servidores Linux implica tareas como:

- instalar servicios
- monitorear recursos
- aplicar actualizaciones
- gestionar usuarios
- revisar logs
- mantener seguridad

Muchas de estas tareas se realizan desde la **terminal y mediante acceso remoto**.

---

# Idea clave de esta lección

Linux es el sistema operativo dominante en servidores debido a su estabilidad, seguridad y flexibilidad.
