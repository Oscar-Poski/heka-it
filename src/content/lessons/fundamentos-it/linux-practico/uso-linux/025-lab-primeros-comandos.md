# Lab: Primeros Comandos

Prueba estos comandos dentro de la terminal simulada. Sigue el orden para obtener el mismo resultado.

1. `date` muestra la fecha y hora actual del sistema.

```bash
student@heka:~$ date
Wed Mar1810:00:00 UTC2026
```

2. `whoami` indica el nombre del usuario con el que has iniciado sesión.

```bash
student@heka:~$ whoami
student
```

3. `pwd` (print working directory) muestra la ruta del directorio actual. Aquí estás en `/home/student`.

```bash
student@heka:~$ pwd
/home/student
```

4. `ls` lista los archivos y carpetas dentro del directorio actual.

```bash
student@heka:~$ ls
README.txt
```

5. `echo Hola` imprime el texto **"Hola"** en la terminal.

```bash
student@heka:~$ echo Hola
Hola
```

6. `clear` limpia la pantalla de la terminal (no muestra salida visible, solo borra el contenido anterior).

```bash
student@heka:~$ clear
```
